Modlitba za farnosť

== SK ==
Plugin generuje shortcode [modlitba_za_farnost], ktorý čerpá obsah z adresy https://www.abuba.sk/knazi/farnosti.php.
Shortcode môžete vložiť na váš WordPress web (do článku, stránky, widgetu, atď.) a automaticky sa zobrazí modlitba na daný deň.
Pre príklad pozri PK farnosť -> https://pezinok.fara.sk/

== EN ==
This plugin generates a shortcode [modlitba_za_farnost], that takes content (daily prayer) from  url https://www.abuba.sk/knazi/farnosti.php.
You can insert this shortcode to your web and prayer for respected day will be displayed.

== Installation ==

== SK ==
1. Plugin nainštalujete priamo cez WordPress -> Administrácia -> Pluginy -> Pridať nový -> Hľadajte Modlitba za farnosť
2. Prípadne stiahnite si .zip súbor a nahrajte ho cez možnosť "Nahrať plugin" v administrácii. 

== EN ==
1. Upload the full modlitba-za-farnost directory into your wp-content/plugins directory.
2. In WordPress select Plugins from your sidebar menu and activate the Modlitba za farnosť plugin.
